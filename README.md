# 🌌 Loom And Lite

Where Imagination Meets Innovation.

Loom And Lite is a modern art-learning platform built with Next.js, TypeScript, Tailwind CSS, Firebase, Framer Motion, and Three.js.

## ✨ Features

- 🎨 300+ Drawing Tutorials
- 🤖 AI Drawing Assistant
- 🔍 Smart Search
- 📚 Categories
- ❤️ Favorites
- ⭐ Ratings & Reviews
- 💬 Comments
- 📈 Progress Tracking
- 📺 YouTube Tutorials
- 📄 PDF Downloads
- 👤 User Accounts
- 🌙 Dark Space Theme
- 📱 Fully Responsive

## 🛠 Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Firebase
- Framer Motion
- Three.js
- React Query
- Zustand

Currently under active development.
